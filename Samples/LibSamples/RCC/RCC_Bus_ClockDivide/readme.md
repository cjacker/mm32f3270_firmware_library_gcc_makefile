# RCC_Bus_ClockDivide

## Introduction

This demo shows the  RCC's clock output feature.

## Run

Add "AHB_HCLK_freq" to Watch1 in debug mode,  the value is 0x016E3600, which is 24M.

Add"APB1_PCLK1_freq" to Watch1 in debug mode,  the value is 0x005B8D80, which is 6M.

Add"APB1_PCLK1_freq" to Watch1 in debug mode,  the value is 0x00B71B00, which is 12M.
