# TIM2_BasicExample

## Introduction

This demo shows the TIM2's basic feature. 
Configure TIM2 timing 1ms, and enable the update interrupt.

## Run

The serial port debugging assistant prints data at 1000ms intervals as follows:
Time=0x0s
Time=0x1s
Time=0x2s
Time=0x3s
Time=0x4s
Time=0x5s 
......
